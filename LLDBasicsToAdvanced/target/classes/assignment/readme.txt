Food Ordering and Delivery Platform
Functional requirements
1. Customer Should be able to browse Restuarant menu and cutomise a menu item (add toppings etc)
2. Customer should be able to place Ordering
3. Order should be validated
4. Assigning a delivery partner
5. Order should be delivered to Customer
6. Notify Customer at each step



Core Entities
1. Customer
2. Restuarant
3. menu
4. DeliveryPartner
5. Order
6. Notify
7. Payment
